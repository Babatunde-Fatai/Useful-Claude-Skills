# Paystack Quick Reference

- API base: `https://api.paystack.co`
- Secret key env: `PAYSTACK_SECRET_KEY`
- Public key env: `PAYSTACK_PUBLIC_KEY`
- Webhook header: `x-paystack-signature`
- Amount unit: smallest currency unit (kobo/pesewas/cents)

Use this file for rapid setup checks. For complete implementation examples, read `one-time-payments.md` and framework-specific guides.
