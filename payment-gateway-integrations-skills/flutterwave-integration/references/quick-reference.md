# Flutterwave Quick Reference

- API base: `https://api.flutterwave.com/v3`
- Backend secret env: `FLW_SECRET_KEY`
- Webhook hash env: `FLW_SECRET_HASH`
- Primary reference key: `tx_ref` (generated by your backend)
- Amount unit: smallest currency unit

Use this file for setup checks. Use implementation references for full flow details.
